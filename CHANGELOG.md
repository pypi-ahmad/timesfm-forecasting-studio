# Changelog

## [0.2.0] - 2026-07-23

### Added

- Guided TimesFM inference workflow for loading, analyzing, forecasting, evaluating, and exporting.
- Comprehensive time-series EDA with quality, distribution, trend, seasonality, ACF, STL, ADF, and robust outlier diagnostics.
- Rolling-origin backtesting with point, probabilistic, interval, latency, and throughput metrics.
- Quantile-based forecast anomaly detection.
- TimesFM XReg forecasting with dynamic numerical and categorical covariates.
- Forecast CSV, interactive HTML, vector PDF, and complete ZIP report downloads with provenance manifests.
- Five high-quality manual test datasets and a reproducible download script.

### Fixed

- Clear loaded datasets now resets the uploader widget and all dataset-derived session results.

### Changed

- Added pinned CPU-compatible dependencies for XReg, statistical diagnostics, and PDF reports.
